# bilibli


## 開発環境

[pipenv](https://pipenv-ja.readthedocs.io/ja/translate-ja/)仮想環境で構築できる。

Python環境にpipenvをインストールする。

```
pip install pipenv
```

pipenvがインストールされたPython環境下で環境構築する

Pipenvパスが通っている場合
```
pienv install
```

Python環境のパスが通っている場合
```
python -m pipenv install
```

pyコマンドのパスが通っている場合
```
py -m pipenv install
```
